¿Qué es el Diseño web responsivo? ¿En qué consiste? ¿Cómo funciona? Diferencias entre el Diseño web responsivo y el Diseño web adaptativo Ventajas y desventajas

Ventajas

-Da a su sitio web una clasificación más alta (SEO):
El diseño de su sitio web responsivo otorga una clasificación más alta a su sitio mientras busca cualquier contenido a través de motores de búsqueda como Google.
-Más fácil de mantener:
Múltiples sitios web para múltiples dispositivos significa que cualquier cambio debe realizarse en todos ellos. Con un sitio web responsivo significa que solo tiene que hacer el cambio una vez.
-Coherencia en diseño y marca:
Un sitio web responsivo mantendrá el mismo diseño en todos los dispositivos.
-Atrae a un público más amplio:
La audiencia accederá a su sitio web desde una variedad de dispositivos diferentes en todas las formas y tamaños. El diseño responsivo los acomodará sin importar cuán grande o pequeña sea su pantalla.

Desventajas

-La navegación se vuelve más difícil:
La calidad de navegación en la pantalla pequeña del dispositivo móvil es diferente de lo que vemos en la pantalla grande del escritorio.
-Ofrece una mala experiencia de usuario:
En general, un sitio web responsivo tarda más tiempo en mostrar los datos en la pantalla y el usuario tiene que esperar para obtener la información a tiempo. Esto irrita al usuario al darle una mala experiencia de navegación.
-Toma más tiempo:
El sitio web responsivo tarda más tiempo en descargarse en la pantalla pequeña.
-Consume mucho tiempo en el desarrollo:
Diseñar un sitio web normal es bastante fácil y consume menos tiempo en comparación con el diseño de un sitio web responsivo.
