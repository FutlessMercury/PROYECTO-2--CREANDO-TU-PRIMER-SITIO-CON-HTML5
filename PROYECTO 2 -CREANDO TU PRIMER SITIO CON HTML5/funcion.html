¿Cómo funciona?

Usualmente un sitio web esta conformado por un conjunto de archivos (los archivos de una página de inicio, página de información, etc.). Cada archivo contiene código HTML y contenido (como texto e imágenes). Las páginas web actualmente se diseñan con archivos llamados hojas de estilo en cascada (CSS), entonces un sitio web responsivo se encarga de aplicar un conjunto alternativo de archivos CSS según el dispositivo que sea utilizado para acceder a determinado sitio. Por lo cual sitio se ve y 'responde' de manera diferente según el dispositivo en donde se visualice dicha web.
